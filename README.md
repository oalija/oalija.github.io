# Proyectos de chorradas

Bienvenido a mi página de proyectos.

## Proyectos

- [**PixelPaint (Colorear)**](./colorear/): Una aplicación web interactiva que convierte tus fotos en un lienzo de "pintar por números". Puedes subir cualquier imagen y la app extraerá una paleta de colores y generará las regiones numeradas para que puedas colorearla.
